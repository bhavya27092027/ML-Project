# ML-Project
An end-to-end ML project for recognizing handwritten digits (0–9) using the MNIST dataset. Features data preprocessing, visualization, model training, and evaluation. Implements deep learning with Keras and classical algorithms with Scikit-learn, including accuracy reports and confusion matrix analysis.

This project focuses on recognizing handwritten digits (0–9) using the MNIST dataset. It combines deep learning models built with Keras and classical ML algorithms from Scikit-learn to compare performance. The workflow includes data preprocessing, visualization, model training, and evaluation.

Features:- 

Data preprocessing and normalization

Visualization of MNIST digits

Neural network implementation using Keras (TensorFlow backend)

Classical ML algorithms with Scikit-learn (e.g., Logistic Regression, SVM)

Model evaluation with accuracy, confusion matrix, and classification report

Tech Stack:-

Python

TensorFlow (with Keras API)

Scikit-learn

Matplotlib / Seaborn (for visualization)

NumPy(for data handling)
